# Pitfalls Research - Jaga Warnet (Roblox)

## Pitfall 1 - Spawn chaos tidak terkontrol

- **Warning signs**: Terlalu banyak NPC aktif, FPS drop, event bertabrakan.
- **Prevention**: Batasi max active NPC per kategori + cooldown global spawn.
- **Phase mapping**: Phase 2 (NPC system) dan Phase 5 (balancing).

## Pitfall 2 - Ekonomi tidak seimbang

- **Warning signs**: Pemain kaya terlalu cepat atau selalu bangkrut.
- **Prevention**: Gunakan reward curve per mini-game + biaya harian baseline.
- **Phase mapping**: Phase 1 (economy core) dan Phase 5 (tuning).

## Pitfall 3 - Event anomali terasa random tanpa konteks

- **Warning signs**: Pemain tidak paham kenapa kalah/kena penalty.
- **Prevention**: Telegraphing lewat UI warning + audio cue + objective jelas.
- **Phase mapping**: Phase 3 (mini-game + anomaly).

## Pitfall 4 - Easter egg terlalu obscure

- **Warning signs**: Hampir tidak ada pemain menemukan easter egg.
- **Prevention**: Beri hint halus via NPC dialog atau lingkungan.
- **Phase mapping**: Phase 4 (ending flows).

## Pitfall 5 - Monetisasi merusak fairness

- **Warning signs**: Booster terlalu kuat dan mematikan challenge inti.
- **Prevention**: Batasi monetisasi pada QoL/cosmetic/soft boost.
- **Phase mapping**: Phase 6 (UI + monetization).
