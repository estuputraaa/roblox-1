# Features Research - Jaga Warnet (Roblox)

## Table Stakes

- Loop harian yang jelas (hari berjalan, objective harian, konsekuensi gagal).
- NPC hidup di dunia (spawn/despawn, roaming, interaksi sederhana).
- Economy loop dasar (income, expense, reward dari aktivitas).
- UI status utama (hari, uang, objective, event warning).
- Feedback audiovisual saat event penting (anomali muncul, mini-game sukses/gagal).

## Differentiators

- NPC meme Indonesia dengan animasi lucu dan dialog khas.
- Event anomali horor lokal dengan twist komedi.
- Ending alternatif berbasis trigger unik (easter egg path).
- Mini-game tematik warnet yang saling memengaruhi (mie + repair + anomaly response).

## Anti-Features (v1)

- Open world terlalu besar dengan kendaraan kompleks.
- Crafting/inventory RPG penuh.
- Sistem rank kompetitif global.

## Complexity Notes

- **Low**: UI HUD dasar, placeholder monetisasi
- **Medium**: Economy balancing, mini-game loops
- **High**: Weighted spawn + behavior orchestration + anomaly event pacing

## Dependencies

- NPC behavior butuh data config stabil sebelum tuning spawn rates.
- Ending system bergantung pada event flags dari mini-game dan anomaly events.
- Monetisasi harus non-intrusif dan tidak merusak core loop harian.
